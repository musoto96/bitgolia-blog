Creating a bootable disk.

We will use CAINE GNU/Linux OS in order to create and extract an image from the hard disk. Head over to the official website and download the latest version. Make sure to check the MD5 or the SHA256 hash of your download against the recorded hash. E.g. If you are using Windows you can use the program certutil.exe on the command-line/Powershell: 

PS C:\Users\msoto\Downloads> certutil -hashfile caine13.0.iso SHA256
SHA256 hash of caine13.0.iso:
6d25180757d6a8a71e98706009d7a9ba3613131727fc96c2037d78bbd4c8ce3a
CertUtil: -hashfile command completed successfully.
If it doest match make sure you are not compaing MD5 against SHA256 or that you are comparing against a different version of the download.

Use Rufus to create bootable USB drive using the Caine iso image.
